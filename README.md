# HTML e CSS para Desenvolvimento Web

Repositório dedicado aos meus estudos de **HTML5 e CSS3**, com base em materiais e aulas do professor **Gustavo Guanabara** e outros conteúdos complementares.

O objetivo deste repositório é consolidar conceitos fundamentais de desenvolvimento web, registrando exemplos práticos, anotações e exercícios aplicados durante o aprendizado.

## Resumo do Conteúdo

Além dos exercícios e práticas, também desenvolvi meu próprio resumo com base nos vídeos e materiais disponibilizados, organizando tudo em um documento Word: <a href="https://docs.google.com/document/d/1AdOClTkWyqT1IYin8wU6Mc3vgj8SRs5iMA-JzslLS3A/edit?usp=sharing" target="_blank" rel="noopener noreferrer">Acessar</a>

## Conteúdos Estudados

- Estrutura e semântica do **HTML5**
- Formatação de textos e uso de tags semânticas
- Listas e links (hipertexto)
- Inserção de imagens, áudios e vídeos
- Conceitos de **CSS** (inline, internal e external)
- Cores (RGB, HSL e círculo cromático)
- Tipografia e fontes externas
- Seletores personalizados, pseudo-classes e pseudo-elementos
- Modelo de caixas (Box Model)
- Fundamentos de **Git e GitHub**
- Elementos e validações em formulários (`input`, `button`, `pattern`, `minlength`, etc.)

## Tecnologias Utilizadas**

- HTML5  
- CSS3  
- Git / GitHub  
- Visual Studio Code  

## Objetivo

Desenvolver uma base sólida em front-end para evolução futura em projetos web, aplicações completas e integração com sistemas back-end.
